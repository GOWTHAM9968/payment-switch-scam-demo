
# 💳 UPI QR Scam Simulator

A cybersecurity awareness project that demonstrates how UPI QR Code scams work in real-world social engineering attacks.

## 📌 Project Overview

The UPI QR Scam Simulator is an educational web application designed to help users understand how fraudsters trick victims into making payments while pretending to send refunds or rewards.

This project simulates a common UPI QR scam scenario and explains each step of the attack process in a safe learning environment.

> ⚠️ This project is developed for educational and cybersecurity awareness purposes only.

---

## 🚀 Features

### 👤 Attacker Setup
- Enter attacker UPI ID
- Select scam type
- Set fake refund amount
- Generate malicious QR code simulation

### 📷 QR Code Generation
- Creates a realistic payment QR code
- Demonstrates how attackers disguise payment requests
- Shows victim-facing QR content

### 🔍 Scanner Simulation
- Simulated UPI QR scanner interface
- Demonstrates victim scanning process
- Shows how users can misinterpret payment requests

### 🎯 Scam Workflow Visualization
Step-by-step explanation of:

1. Attacker creates QR code
2. Fake refund/reward claim
3. Victim scans QR code
4. UPI app opens payment screen
5. Victim misunderstands transaction
6. Victim enters UPI PIN
7. Money gets transferred to attacker

### 🎨 Modern UI
- Responsive design
- Interactive interface
- Realistic scanner simulation
- Cybersecurity awareness theme

## 🔐 Cybersecurity Awareness Tips

✅ Verify payment details before entering PIN

✅ Remember: Entering UPI PIN authorizes payment

✅ Receiving money never requires UPI PIN

✅ Double-check merchant and UPI ID

✅ Be cautious of refund and reward scams
